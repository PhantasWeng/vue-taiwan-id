# Guide
## Introduction
A Vue wrap for [taiwan-id-validator2](https://github.com/doggy8088/taiwan-id-validator2)

## Getting Started
### Installation

#### NPM / YARN

```bash
npm install vue-taiwan-id
```

```bash
yarn add vue-taiwan-id
```

#### Import
Import to `main.js`

```js
import vueTaiwanId from 'vueTaiwanId'
Vue.use(vueTaiwanId)
```
